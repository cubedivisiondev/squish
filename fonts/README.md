# Fonts

Three typefaces, all under the SIL Open Font License, Version 1.1. The full license text for each is included in this directory.

| File(s) | Font | Weights | License |
| --- | --- | --- | --- |
| `Jost-400.woff2`, `Jost-700.woff2` | Jost | 400, 700 | [OFL-Jost.txt](OFL-Jost.txt) |
| `Manrope-400.woff2`, `Manrope-700.woff2` | Manrope | 400, 700 | [OFL-Manrope.txt](OFL-Manrope.txt) |
| `SpaceMono-400.woff2`, `SpaceMono-700.woff2` | Space Mono | 400, 700 | [OFL-SpaceMono.txt](OFL-SpaceMono.txt) |

Jost sets the headings and the SQUISH wordmark. Manrope sets the body. Space Mono sets the monospace UI.

The `.woff2` files are Latin subsets built from the OFL sources distributed via [Fontsource](https://fontsource.org). Under the OFL, the Reserved Font Names may not be used to promote modified versions.
